# staticGoogleMap

Draws a static Google Map.

## Usage:
```
<img src="{{ staticGoogleMap({
    center: 'Brooklyn+Bridge,New+York,NY',
    maptype: 'roadmap',
    zoom: 13,
    size: '900x900',
    key: 'YOUR_API_KEY'
}) }}" alt="">
```
